# Kirjat

Sovelluksessa käyttäjät pystyvät jakamaan lukemiaan kirjoja. Kirjan tiedoissa lukee kirjailija.

## Sovelluksen asennus

Asenna flask-kirjasto:

+ $ pip install flask


Voit käynnistää sovelluksen näin:

+ $ flask run
